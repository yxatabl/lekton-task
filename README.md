## LEKTON TASK

### Описание решения
Консольное приложение для управления библиотекой. Все данные храняться в оперативной памяти, реализованны функции добавления, удаления, просмотра и поиска книг, а также вывод всей статистики.

Реализованы авто-тесты всего функционала. Добавлена автоматическая сборка в jar пакет.

### Запуск
1. Скачать jar из releases
2. Запустить ```java -jar LibraryApp.jar```
