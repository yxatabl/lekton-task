## Описание решения
Все данные храняться в оперативной памяти, реализованны функции добавления, удаления, просмотра и поиска книг, а также вывод всей статистики.

Созданы авто-тесты всего функционала. Добавлена автоматическая сборка в jar пакет.

## Запуск
1. Скачать [релиз](https://github.com/yxatabl/lekton-task/releases/latest)
2. Запустить ```java -jar LibraryApp.jar```
